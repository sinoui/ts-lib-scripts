# eslint-config-ts-lib

[ts-lib-scripts](https://github.com/sinoui/ts-lib-scripts) 创建的 ts 库项目使用的 ESLint 配置

## 安装

```bash
yarn add eslint-config-ts-lib typescript --dev
```

## 使用

```json
{
  "eslintConfig": {
    "extends": "ts-lib"
  }
}
```
